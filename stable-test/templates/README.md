# Templates

Edit these files to control shared UI and page backgrounds:

- `templates/ui-base.css`: shared menu + uitleg sheet tokens (all pages)
- `templates/main-index.css`: main index background bands
- `templates/cards-index.css`: cards index background

Per set overrides are done via `sets/<set>/meta.json` using `ui.vars` (CSS variables)
so you can override any of the variables above per set.
